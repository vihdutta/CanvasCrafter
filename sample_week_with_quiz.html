Module 1 Learning Objectives: Particle Kinetics

  • LO1: Understand the relationship between position, velocity, and acceleration
  • LO2: Select appropriate coordinate systems and be able to decompose them correctly (cartesian, polar/cylindrical, normal-tangential)
  • LO3: Execute coordinate transformations
  • LO4: Define kinematic variables using vector notation
  • LO5: Solve systems of linear equations
  • LO6: Analyze kinematic equations that require implementation of calculus
